## 📌 Project Overview

This project predicts house prices using Machine Learning techniques. The model is trained on the Boston Housing dataset and compares multiple regression algorithms to identify the best-performing model.

The project includes:

- Data Cleaning
- Missing Value Handling
- Exploratory Data Analysis (EDA)
- Data Visualization
- Feature Engineering
- Model Training
- Model Evaluation
- Price Prediction
- Model Saving using Joblib

# 📊 Dataset Features

The dataset contains the following important attributes:

| Feature | Description |
|----------|-------------|
| CRIM | Crime Rate |
| ZN | Residential Land |
| INDUS | Industrial Area |
| CHAS | Charles River |
| NOX | Nitric Oxide |
| RM | Average Rooms |
| AGE | House Age |
| DIS | Distance to Employment Centers |
| RAD | Highway Accessibility |
| TAX | Property Tax |
| PTRATIO | Student-Teacher Ratio |
| LSTAT | Lower Status Population |
| MEDV | House Price (Target) |

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/House-Price-Prediction.git
```

Move into project directory

```bash
cd House-Price-Prediction
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

# ▶ Run Project

### Step 1

Perform Data Cleaning

```bash
python eda.py
```

### Step 2

Train Machine Learning Model

```bash
python train.py
```

### Step 3

Predict House Price

```bash
python predict.py
```
